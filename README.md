# 声音监测网站
> 用来测试当前用户身边环境的噪音，可以利用手机的麦克风来收集当前环境的声音，并通过一系列算法，得到噪声具体分贝的大小，并以图文的方式来显示。
收集所有用户的噪音数据生成全省/全国的噪音分布图，有利于政府对噪音污染的整治与处理。该软件还会推荐附件的低噪音景区，有利于相关景区的发展。
